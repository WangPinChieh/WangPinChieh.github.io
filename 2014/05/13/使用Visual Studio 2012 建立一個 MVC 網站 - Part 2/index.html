使用Visual Studio 2012 建立一個 MVC 網站 - Part 2


新增ㄧ個 Controller

首先，我們在 Controller 資料夾上按右鍵，新增ㄧ個 Controller



接下來我們要替這個Controller命名，我們可以看到它幫我們自動把Default1選起來，後面自動加上Controller，這是MVC命名的方式，我們只可以改動前面的部份，若連後面都改動，會導致找不到網頁!
切記後面的Controller別去做刪除或修改。



點下新增後，我們會看到下面的程式碼:

HomeController 這個 Class 包含了ㄧ個Index()方法，回傳 ActionResults 型別的値，這裡有一點要特別注意的是，Controller 負責告訴 ASP.NET MVC Framework 下ㄧ個要做甚麼(What to do)，而不是下ㄧ個要如何做(How to do)。

舉例來說，若ㄧ個 Controller 要顯示ㄧ個 View，這個 Controller 會把ㄧ個 ViewResult 丟給 View，讓 View 去產生給使用者看的頁面，而不是 Controller 去產生給使用者看的頁面。


新增ㄧ個View


接下來，我們在Index() 上按右鍵，選到 Add View… 來新增ㄧ個View。


完成後，我們可以在Views > Home > Index.cshtml找到我們剛剛加入的View



我們可以點進去編輯HTML，完成我們的第一頁!
